# Wirtualna Pizzeria - Programowanie Obiektowe 🍕

Prosty skrypt w PHP pokazujący, jak działa programowanie obiektowe (klasy i obiekty).

## Co to robi?
* **Klasa Pizza** – tworzy pizzę (nazwa, sos, ciasto, składniki) i sama oblicza jej cenę na podstawie grubości ciasta i liczby dodatków.
* **Klasa Zamowienie** – pozwala stworzyć zamówienie, dorzucić do niego wybrane pizze i zmieniać jego status (np. na "Zakończony").
* **Klasa listaZamowien** – zbiera wszystkie zamówienia w jedno miejsce i pozwala je filtrować (np. wyświetla tylko nowe albo tylko usunięte).

## Jak uruchomić?
Projekt nie potrzebuje bazy danych. Po prostu odpal plik `pizza.php` w XAMPPie lub dowolnym testerze PHP online.
